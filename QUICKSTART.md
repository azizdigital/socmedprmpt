## First Time Using GitHub?

### Easy Upload Method:

1. Go to https://github.com/new
2. Create repository (name: `prompt-posting-generator`)
3. On the empty repository page, click "uploading an existing file"
4. Drag ALL folders and files into the upload area:
   - index.html
   - manifest.json
   - sw.js
   - css/ (entire folder)
   - js/ (entire folder)
   - icons/ (with the 2 PNG files you generated)
5. Scroll down, click "Commit changes"
6. Go to Settings â†’ Pages
7. Enable Pages (main branch, root folder)
8. Done!
